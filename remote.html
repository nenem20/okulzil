🔒 HTTPS Kurulumu — Mikrofon ve Güvenli Bağlantı İçin
Adım 1 — Sertifikayı İndirin
Adım 2 — Sertifikayı Yükleyin
Windows (Chrome/Edge): İndirilenler → zil_cert.crt'ye çift tıkla → Sertifikayı Yükle → Yerel Makine → İleri → Tüm sertifikaları aşağıdaki depolama alanına yerleştir → Gözat → Güvenilen Kök Sertifika Yetkilileri → Tamam → İleri → Son

🦊 Firefox kullanıyorsanız — sertifikayı yükledikten sonra aşağıdaki butona da basın:
İndirilen .bat dosyasına çift tıklayın → Firefox'u kapatıp açın → HTTPS'e geçin.

Android: İndirilenler → zil_cert.crt'ye dokun → "CA sertifikası" seç → onayla
iPhone/iPad: zil_cert.crt'ye dokun → Ayarlar → Genel → VPN ve Aygıt Yönetimi → Profili Yükle → Güven ver
